So, [null is not an object](https://wtfjs.com/wtfs/2010-02-12-null-is-not-an-object) though sometimes null is a Number.

```
    isNaN( null ); // false
    null === NaN; // false
    null == NaN; // false
```

isNaN converts null to number.

<code>
  Number( null ); // 0
</code>
