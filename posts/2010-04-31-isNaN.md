So, [null is not an object](http://wtfjs.com/2010/02/12/null-is-not-an-object) though sometimes null is a Number.

```
isNaN( null ); // false
null === NaN; // false
null == NaN; // false
```

isNaN converts null to number.

```
Number( null ); // 0
```
